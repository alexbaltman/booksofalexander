Template Variables

Template variables are names that get replaced with dynamic values that the name represents. You can use template variables in emails and in the templates of web pages. To use a variables just write the name of the variable enclosed in {{'{{'}} and {{'}}'}} if you want to use it in a web page template or with a single { and } if you want to use it in an email template (for example {name})

Name Description Locations where this variable is available
shipping_label_url The URL to the image of the shipping label that a book seller hast to print On the page after the checkout and in the email that gets sent after the checkout
html_payment_method_info The info that tells the seller what he or should do regarding the selected payment method. The text comes from the specific field on the content setings page (for example from the 'PayPal Checkout (HTML)' field). This string is formated as HTML. On the page after the checkout and in the email that gets sent after the checkout
text_payment_method_info The same as the last one but not formated as HTML On the page after the checkout and in the email that gets sent after the checkout
html_exchange_method_info The same as 'html_payment_method_info' but for the exchange method On the page after the checkout and in the email that gets sent after the checkout
text_exchange_method_info The same as 'text_payment_method_info' but for the exchange method On the page after the checkout and in the email that gets sent after the checkout
html_carrier_info The same as 'html_payment_method_info' but for the carrier On the page after the checkout and in the email that gets sent after the checkout
text_carrier_info The same as 'text_payment_method_info' but for the carrier On the page after the checkout and in the email that gets sent after the checkout
